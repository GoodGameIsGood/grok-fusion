# Style Recipes (on demand)

Load **at most one** recipe when the brief is vague and no existing DS exists. Do not stack recipes. Skip entirely when brownfield tokens already define the look.

## 1. Atelier material

Quiet luxury: tactile surfaces, restrained palette (ink, chalk, one warm accent), serif display used sparingly, generous whitespace, photography-led hero. Avoid cream+#F4F1EA+terracotta default cluster unless the brief asks for it — pick a distinct warm/cool pair from the subject.

## 2. Engineered quiet

Product/tool feel: precise grid, monospace metadata accents, sharp or near-zero radius, high-contrast type, minimal motion. No neon cyberpunk glow kits. Accent from product brand, not purple-indigo defaults.

## 3. Editorial field

Content-led: strong display type, asymmetric columns, full-bleed imagery, short captions. Not broadsheet hairline pastiche — use intentional rules only where they encode structure.

## How to apply

1. Name the recipe in the token preflight “signature” line
2. Map recipe → tokens (colors, type, radius, motion)
3. Implement; if output drifts toward anti-slop clusters, revise tokens first
